📁 File types

  • .doc - Word Documents (protocols, reports)
  • .pdf - Portable Document Format (reports, protocols, correspondence)
  • .xls - Excel Spreadsheets (older format)
  • .xlsx - Excel Spreadsheets (subject lists, data tables)
  • .xsl - XSL Stylesheets (data formatting templates)
  • .xml - Structured Data Files (clinical trial metadata)
  • .xpt - SAS Transport Files (statistical analysis datasets)

Most FOIA releases are statistical datasets (.xpt) and PDF documents, with Excel files typically containing participant lists or summary data.
